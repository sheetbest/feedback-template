About this template

This free feedback form lets you collect responses directly in Google Sheets using the Google Sheets API powered by SheetBest.

You can find the source code on GitHub on the feedback template repository. You can also check out the sample Google Sheet that receives the form responses (just duplicate it if you want to use it).
